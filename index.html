Qubob

Qubob is example usage of django-bob library for Django https://bitbucket.org/thesheep/bob/

How to run?

Virtualenv (optional)

python -m virtualenv --no-site-packages ~/qubob
cd qubob/

Required:

git clone https://github.com/quamilek/qubob.git
source bin/activate
cd qubob/
pip install -e .
cd src
./manage.py runserver

And open this link in browser: http://127.0.0.1:8000
